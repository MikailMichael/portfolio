Mixed messages

A program that runs in JavaScript that will randomly generate multiple strands of DNA for an imaginary organism known as pAequor. It saves each organism generated to a uniqure specimen ID and keeps randomly generating specimens until it collects an array of 30 that could survive the environment based on their DNA composition; it then prints the list of specimen IDs that would survive. Then it randomly compares 2 of the surviving specimens to each other and prints the percentage of DNA they have in common, repeating 10 times.

  • Visual Studio Code - IDE
